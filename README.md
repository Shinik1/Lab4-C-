Задание 6. Перегрузка операций
Поля:
hours (0-23)
minutes (0-59)

Конструктор:
Принимает часы и минуты
Выбрасывает исключение при недопустимых значениях

Методы:
ToString() - возвращает время в формате "HH:MM"
InputTime() - статический метод для ввода времени с консоли с валидацией

Арифметические операции:
operator + - сложение времен с циклическим переходом через полночь
operator - - вычитание времен с циклическим переходом через полночь

Автоматическая нормализация времени (1440 минут в сутках)
Циклическая арифметика: результат всегда в диапазоне 00:00-23:59
Защита от некорректного ввода в методе InputTime()
